CryptoTooL
简单的加解密工具，仅支持凯撒密码和XOR密码
CryptoToolProject/          
├── CMakeLists.txt          
├── cryptotool.cpp          
├── README.md               
└── build/                  
    └── CryptoTool          

如何构建与运行项目：
1.
cd CryptoToolProject
2.
mkdir build && cd build
3.
cmake ..
4.
make
5
./CryptoTool
